# 更新日志

版本号在设置面板底部可以看到（⚙️ → 拉到最下面）。每次更新用
`node scripts/bump-version.js x.y.z` 同步升级版本号，手机上的旧缓存就会自动失效、换成新版本。

## 如何回滚到旧版本

**方法一（最简单，几秒钟）**：去 [Vercel 项目页](https://vercel.com/adrianwkh10macs-projects/brain-gym) →
Deployments，找到想恢复的那次部署 → 右边 "..." → **Promote to Production**。
线上网址不变，立刻切回旧版本，随时可以再切回来。

**方法二**：告诉我"回滚到 v1.0.0"这样的话，我会切到对应的 git tag，重新推送部署。

每个版本都会打一个 git tag（如 `v1.1.0`），GitHub 上能看到完整历史。
