# 🧠 大脑体操馆

9 合 1 脑力训练游戏合集：舒尔特方格、火眼金睛、记忆翻牌、数字华容道、迷宫、数独、数织、彩虹连线、皇后谜题。

- ♾️ **无限关卡**：所有题目由程序即时生成（数独/数织/皇后保证唯一解或纯推理可解），永远玩不完
- 🎯 **双模式**：练习模式自选难度（最高「疯狂」档）+ 无限闯关模式（一关比一关难）
- 👁️ **护眼**：标准 / 护眼暖色 / 夜间三种主题，每玩 20 分钟提醒休息 20 秒
- 📱 **可安装到手机**（PWA）：装好后全屏运行、离线可玩，跟真 App 一样
- 💾 成绩自动保存在本机

## 电脑上玩

直接双击 `index.html` 就能玩（此时不启用离线缓存，不影响游戏）。

## 部署到网上 + 安装到 Android 手机

### 第一步：上传（推荐 Netlify Drop，免费、不用注册命令行）

1. 打开 <https://app.netlify.com/drop>（需要注册/登录一个免费账号）
2. 把整个 `brain-gym` 文件夹拖进网页中间的虚线框
3. 十几秒后你会得到一个网址，形如 `https://xxxx.netlify.app`

> 备选方案：GitHub Pages —— 把本文件夹推到 GitHub 仓库，Settings → Pages → 选择分支即可。

### 第二步：安装到手机

1. 在 Android 手机的 **Chrome** 里打开上面的网址
2. 点右上角「⋮」菜单 → **「添加到主屏幕」**（或弹出的「安装应用」提示）
3. 桌面上就会出现「大脑体操」的图标，点开全屏运行，**以后没网也能玩**

> iPhone 同理：Safari 打开 → 分享按钮 → 「添加到主屏幕」。

## 说明

- 这是 PWA（渐进式网页应用），不是 APK 安装包，但安装后的体验与原生 App 基本一致
- 更新游戏后重新拖到 Netlify Drop 同一个站点即可，手机端下次联网打开会自动更新
- 成绩存在手机本机（localStorage），卸载或清浏览器数据会清空
